Okta Lab Setup

Status: Scaffold — content in progress

Requirements​

  • Okta Developer account (free): developer.okta.com
  • Org domain: dev-XXXXXXX.okta.com

Initial Configuration​

  1. Create test users in Okta Universal Directory
  2. Set up Okta Verify push factor (required for MFA fatigue simulation)
  3. Configure a SAML app integration pointing to a test SP
  4. Enable Okta System Log streaming to SIEM

Attack Tool Setup​

# Install pyokta or use the Okta API directly
pip install okta

# Set OKTA_API_TOKEN env var
export OKTA_API_TOKEN="your-token-here"
