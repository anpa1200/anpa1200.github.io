ThreatMapper Web — MITRE ATT&CK CTI Workspace

Browser-native CTI workbench for exploring MITRE ATT&CK, building TTP layers, comparing group profiles, identifying detection gaps, and exporting analyst-ready outputs.

This public Web version runs in the browser and is intended for exploration. It does not perform AI report extraction, does not upload private reports to a backend, and does not use an LLM service. Use the Docker deployment for private AI-assisted report analysis.
