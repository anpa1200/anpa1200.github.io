Threat Matrix — Public ATT&CK Workspace for AdversaryGraph

Browser-native CTI workbench for exploring MITRE ATT&CK, building TTP layers, comparing group profiles, identifying detection gaps, and exporting analyst-ready outputs.

Threat Matrix is a public, read-only browser workspace associated with the wider AdversaryGraph platform; it is not the full self-hosted product. AdversaryGraph adds AI-assisted CTI report analysis, stored investigations, IOC enrichment, OpenCTI/STIX/TAXII/MISP workflows, detection engineering handoff, API automation, and MalwareGraph-backed malware analysis.

Attack Simulation was introduced in the historical v5.0.0 release and is included in the current stable AdversaryGraph v6.0.0 release.

Analyst-assistance disclaimer: TTP overlap, actor similarity, generated summaries, and coverage findings are investigation leads. They do not prove attribution, exploitability, detection quality, or operational readiness without analyst validation and source evidence review.
