Malware Analysis Category

The current platform includes a MalwareGraph-backed workflow for controlled Windows sample research. It is intended for isolated lab use, analyst validation, and defensive CTI/detection work, not automatic malware verdicts.

V4 Malware Analysis: Static Triage To AI Debug Summary

Level: Malware Analysis

Create a case, upload a raw Windows sample or password-protected ZIP, review hashes and PE facts, inspect strings and IOCs, run unpacking checks, load decompilation/debug views, step through functions, and produce an AI-assisted function and behavior summary.

Output: Clickable IOCs, ATT&CK TTPs, suspicious functions, unpacking evidence, debug notes, validation gaps, and an analyst-ready malware summary linked back into AdversaryGraph investigation workflows.
